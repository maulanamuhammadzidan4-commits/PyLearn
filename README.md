## Day 1: Python Syntax
Belajar sintaks dasar: output, variabel, & komentar.
Belajar tipe data: string, number, dan boolean

```python
# komentar
c = "Hello World"
print(c)
```

**Materi:**
* Output: Menggunakan statement print
* Variabel: Cukup menuliskan nama variabel lalu diikuti data
* Komentar: Diawali dengan simbol tag/pagar (#)
* String: Data untuk teks
* Number: Data untuk angka
* Boolean: Logika dasar (benar/salah)
